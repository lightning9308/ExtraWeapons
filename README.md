## ExtraWeapons

ExtraWeapons adds **19** unique weapons, armor, and items.
Perfect for SMP servers and RPG servers.

![Player using the AntiFlight Stick on a flying bee](https://cdn.modrinth.com/data/cached_images/e0aa621e95d6a2b9732c87f92183a27e8485e46e.gif)

### Features
- Custom weapons
- Custom armor
- Custom items
- Configurable stats

## Commands

| Command |Permission| Description |
|-|-|-|
|`/extraweapons get <item>`|extraweapons.admin|Get custom item|
|`/extraweapons reload`|extraweapons.admin|Reload configs (right/left click held item to update lore)|
|`/extraweapons menu`|extraweapons.admin|Opens a menu containing all custom items|

### Supported Version
 1.21.x - 26.1.2
